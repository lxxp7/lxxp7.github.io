Alternance chez Dwarf Animation Studio

En tant qu'alternant développeur chez **Dwarf Animation Studio**, je travaille sur des solutions d'automatisation pour le pipeline de production 3D.

Projet NightCrawler

NightCrawler est une solution d'automatisation des rendus nocturnes pour Unreal Engine. Il permet de :

  • Analyser les modifications dans **Perforce**
  • Gérer les dépendances avec **Unreal Engine**
  • Automatiser les rendus via **Deadline**
  • Offrir une interface web pour le suivi des rendus

Technologies utilisées

  • Python (Flask, SQLAlchemy, Celery)
  • JavaScript (Frontend Flask/Jinja2)
  • Unreal Engine (Python scripting)
  • Perforce (gestion des versions)
  • Deadline (gestion des rendus)
