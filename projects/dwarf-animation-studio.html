En tant qu'alternant développeur chez Dwarf Animation Studio, je travaille sur des solutions d'automatisation pour le pipeline de production 3D.

Projet NightCrawler

Le projet NightCrawler vise à créer une solution logicielle d’automatisation des rendus nocturnes pour un pipeline de production créative. Ce service devra intégrer plusieurs éléments techniques tels que Perforce, Unreal Engine, Deadline et une interface utilisateur web afin de garantir une gestion efficace des dépendances et des rendus. L’objectif est de simplifier la gestion des rendus en fonction des modifications apportées dans Perforce, tout en offrant une interface intuitive pour superviser et configurer les différents processus.

Fonctionnalités

  • Analyse des modifications dans Perforce
    • Extraction des changelists depuis Perforce.
    • Identification des fichiers modifiés.
    • Synchronisation du workspace avec les modifications pertinentes.
  • Gestion des dépendances
    • Appel à Unreal Engine via un script Python pour analyser les dépendances.
    • Résolution des Level Sequences associés aux fichiers modifiés.
  • Automatisation des rendus
    • Génération des commandes Unreal Engine pour effectuer les rendus.
    • Suivi des statuts de rendu (état démarré, progression, terminé, erreur).
  • Interface utilisateur web
    • Gestion des configurations :
      • Paramètres de connexion à Perforce.
      • Liste d’inclusion/exclusion basée sur des expressions régulières.
      • Planification des rendus, pour forcer le rendu d’une séquence (cron jobs).
    • Visualisation des logs et statuts des rendus.
    • Tableau des rendus prévus avec la possibilité de déclencher un rendu manuel.
    • Affichage des changelogs pour chaque rendu.
    • Accès via login ?
  • API REST
    • Points d’entrée pour interagir avec les processus de rendu et les configurations.
    • Suivi en temps réel des statuts via des notifications.
  • Gestion des données
    • Stockage des statuts de rendu.
    • Export des changelogs sous forme de fichiers texte associés aux rendus.
  • Mise en place de tests
    • Test unitaire du code.
    • Test fonctionnel avec data de test.

Technologies utilisées

Python (Flask, SQLAlchemy, Celery) JavaScript (Frontend Flask/Jinja2) Unreal Engine (Python scripting) Perforce Deadline
