Stage de 2ᵉ année de BUT

Amélioration du pipeline de production chez Dwarf Animation Studio

Contexte du stage

Ce stage, réalisé au sein de Dwarf Animation Studio, m’a permis de contribuer directement au pipeline de production du studio. Mon travail a porté sur l’implémentation d’une fonctionnalité clé : l'automatisation de la création d’un dossier de LookDev dans Unreal Engine. Ce processus, auparavant manuel, est désormais intégré dans un outil interne du studio, ce qui permet un important gain de temps pour les équipes de production.

Objectifs et réalisation

  • Analyse du pipeline existant et des outils internes développés par le studio
  • Apprentissage de l’API Python d’Unreal Engine pour manipuler des assets et des niveaux
  • Implémentation dans l’application interne "loader" pour automatiser la création du dossier de LookDev
  • Optimisation et nettoyage du code pour améliorer la maintenabilité

Illustrations du travail réalisé

Technologies utilisées

Python Unreal Engine (API Python) Pipeline d'animation 3D

Défis et apprentissages

Ce stage m’a permis de progresser sur plusieurs plans :

  • Compréhension approfondie du fonctionnement d’un pipeline de production en animation 3D
  • Autonomie dans la prise en main d’un outil aussi massif qu’Unreal Engine
  • Collaboration avec les équipes techniques du studio pour s'intégrer dans un environnement de production réel
