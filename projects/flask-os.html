Projet OpenSource Flask

Un template d'API et de frontent open source Flask respectant les bonnes pratiques de programmation. Réalisé dans le cadre de mon apprentissage de Flask au début de mon alternance chez Dwarf Animation Studio.

Aperçu du projet

Ce projet open source, développé avec Flask applique les bonnes pratiques de développement telles que la séparation des préoccupations, et la gestion des erreurs pour offrir une solution flexible et robuste grâce au système de Blueprints.

Lien vers le Repo GitHub

Fonctionnalités clés

Bonnes pratiques de programmation Flask

Implémentation des bonnes pratiques de développement Flask pour les applications modulables à l'aide de l'outil Blueprint permettant de créer différentes versions des applications facilement et de les déployer.

Détails techniques

Technologies utilisées

Python Flask / Jinja2 JavaScript

Implémentation

  • API RESTful pour une communication claire et une architecture extensible
  • Utilisation de Flask comme framework principal, avec des blueprints pour une séparation logique des différentes parties de l'application
  • Gestion des erreurs et des exceptions pour offrir une expérience utilisateur robuste et fiable
  • Tests unitaires et fonctionnels utilisant pytest pour garantir la qualité et la fiabilité du code

Défis et apprentissages

Au cours de ce projet, j'ai approfondi mes connaissances en matière de :

  • Développement d'une architecture Flask modulaire et maintenable
  • Gestion des erreurs et implémentation de solutions robustes pour un environnement de production
